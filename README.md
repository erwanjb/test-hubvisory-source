## Installation

```bash
$ npm install
```

## Building the app

```bash
$ npm run build
```

## Running the app

```bash
$ npm run start
```

## Building and Running the app

```bash
$ npm run full
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
